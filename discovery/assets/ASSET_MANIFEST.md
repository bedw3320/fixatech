## Image Optimization Strategy

### Tools
- **ImageOptim** (Mac) or **Squoosh** (Web)
- **cwebp** (command-line WebP converter)
- **Sharp** (Node.js image processing)

### Process
1. **Convert to WebP:**
   ```bash
   cwebp -q 80 input.png -o output.webp
   ```

2. **Create JPG fallback:**
   ```bash
   convert input.png -quality 80 output.jpg
   ```

3. **Verify dimensions match design spec:**
   - Hero images: 389×497px
   - Product cards: 315×409px
   - Service images: 389×497px

4. **Implement responsive images in Liquid:**
   ```liquid
   <picture>
     <source srcset="{{ 'hero-image.webp' | asset_url }}" type="image/webp">
     <img src="{{ 'hero-image.jpg' | asset_url }}" alt="Hero product" loading="lazy">
   </picture>
   ```
